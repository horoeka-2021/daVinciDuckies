# daVinciDuckies

This is a starter project to illustrate React components consuming a web API.


## Setup

To experience this repo in all of its glory:

* Clone this repo

```sh
cd daVinciDuckies
npm install
npm run dev
```

* Navigate to [http://localhost:3000](http://localhost:3000)
